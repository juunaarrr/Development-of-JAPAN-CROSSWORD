# 🧩 Японские кроссворды 
Это интересная головоломка, в которой пользователю нужно закрасить нужные клетки, используя цифры-подсказки, отображающие количество клеток, закрашенных в строке или столбце подряд. В результате прохождения получается картинка :)

## ✨ Особенности:

- 🎨 **Цветовая индикация** — закрашенные и пустые клетки наглядно выделены
- 🖱️ **Простое управление** — кликайте мышью по клеткам для закрашивания/очистки
- ✅ **Проверка решения** — игра определяет, когда кроссворд разгадан полностью

## 🛠️ Управление: 

- Нажатие ЛКМ по ячейке - клетка заливается чёрным цветов
- Нажатие ПКМ по ячейке - клетка отмечается крестиком
- Если зажать ЛКМ и провести мышкой по полю - клетки будут закрашиваться непрерывно
- Если зажать ПКМ и провести мышкой по полю - клетки будут отмечатьсся крестиком непрерывно
- При неправильной отметке клетки снимается одна жизнь
